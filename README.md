Study Tracker
A simple Python command-line program to track your study sessions and total study time.

Features
-Add a study session with subject and duration.
-View all sessions.
-See total study time in hours and minutes.
-Delete all sessions for a specific subject.

Project Files
-main.py: Handles the command-line menu and user input.
-tracker.py: Functions to add, delete and retrieve sessions.
-storage.py: Manipulates the JSON file.

Notes
-The program works on Windows, macOS and Linux.
-All times are in minutes but displayed as hours and minutes.
